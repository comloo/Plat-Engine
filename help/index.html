TS9 - Brings you a new program to store your games!

Plat Engine is a platform intended to make a place for the gamer to store their favourite games. Plat Engine also provides some different ways that allows the user to view their games by using table view and bar graphs. Users are also allowed to sort their games and search for specific games they played and stored before.

If you have any questions, check out the features on the left.

Before using the program, being a registered user will allow you to have private access to your own game collection.

Have fun! :)
