View

The view contains four seperate functions:

1. Chart View:

The function "chart" will help user to make a chart using the data in their database.

2. Bar Graph View:

The function "Bar graph" helps users to make a bar graph which contains the number games in each genre.

3. Sort

The function "sort" will sort the records by size, name, company, etc.

4. Search

The function "search" will search the records using game name.
